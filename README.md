# Библиотека матричных операций с дробями

Библиотека и тестовое приложение на языке C++ для задания номер 3 по теме
«Дроби и матричные операции».

Проект реализует операции с матрицами на базе класса `Fraction`, который хранит
обыкновенные дроби через целочисленный тип `mpz_t` из библиотеки GMP.

## Функциональность

- хранение и сокращение обыкновенных дробей произвольной длины;
- арифметические операции над дробями;
- сравнение дробей;
- умножение матрицы на вектор;
- классический алгоритм умножения матриц;
- блочный алгоритм умножения матриц;
- алгоритм Штрассена для умножения матриц;
- решение СЛАУ методом Гаусса;
- решение СЛАУ методом Жордана-Гаусса;
- нахождение обратной матрицы методом Гаусса;
- нахождение обратной матрицы методом Жордана-Гаусса.

## Сборка и запуск

Для сборки нужна установленная библиотека GMP.

В Debian ее можно установить командой:

```bash
sudo apt install libgmp-dev
```

Сборка библиотеки и тестового приложения:

```bash
mkdir -p build
g++ -fPIC -c src/fraction.cpp src/matrix.cpp -I include
g++ -shared -o libmatrix.so fraction.o matrix.o -lgmp -lgmpxx
g++ main.cpp -I include -L. -lmatrix -lgmp -lgmpxx -o build/app
```

Запуск:

```bash
LD_LIBRARY_PATH=. ./build/app
```

## Тесты

В `main.cpp` приведены тестовые примеры для всех основных операций:

- умножение матрицы на вектор;
- классическое умножение матриц;
- блочное умножение матриц;
- умножение матриц алгоритмом Штрассена;
- решение системы линейных уравнений методами Гаусса и Жордана-Гаусса;
- нахождение обратной матрицы двумя методами.
